Penjelasan tentang CamPhisher

# Apa itu CamPhish???

CamPhish adalah suatu script untuk mengambil gambar dari kamera depan ponsel target atau kamera web PC hanya dengan mengirimkan tautan.


# Bagaimana mekanisme yang di gunakan CamPhish

CamPhish adalah teknik untuk mengambil gambar kamera depan ponsel atau webcam PC target. CamPhish menghosting situs web palsu di server PHP bawaan dan menggunakan ngrok & serveo untuk menghasilkan tautan yang akan kami teruskan


# Apa aja sih Fitur / opsi pilihannya?...

Dalam alat ini saya menambahkan dua templat halaman web otomatis untuk target yang terlibat di halaman web untuk mendapatkan lebih banyak gambar kamera.

| Opsi pilihan ada 3 jenis web phising yaitu:
  • Festival Wishing
  • Live YouTube TV
  • Online Meeting [beta version]
| Fitur yang di sediakan adalah..
  • memotret target dari kamera depan
  • memberikan informasi browser yang di gunakan target
  • memberikan longtitude dan latitude atau coordinat target berada
  • memberikan alamat ip address handphone target, yang dapat di gunakan untuk menyadap whatsapp target

# Telah di test di platfrom berikut

  • Kali Linux
  • Termux
  • MacOS
  • Ubuntu
  • Parrot Sec OS

# Installasi dan requirements

Script ini memerlukan PHP untuk webserver, SSH atau tautan serveo. Pertama jalankan perintah berikut di terminal Anda


•Command:
$ apt update && apt upgrade
$ apt-get -y install php openssh git wget
| Untuk update package dan install requirements di software Ubuntu dll

•Untuk Termux bisa gunakan command berikut..
$ pkg update && pkg upgrade
$ pkg install php
$ pkg install openssh
$ pkg install git
$ pkg install wget

•Bisa juga di gabung agar memginstall semua requirements sekaligus dengan command seperti berikut..
$ pkg update && pkg upgrade && pkg install php && pkg install openssh && pkg install git && pkg install wget

pergi ke google atau chrome lalu ketik ngrok authtoken, setelah itu login ke ngrok lalu cari ngrok authtoken, dan buat authtoken nya lalu salin dan tempel di terminal


•jika anda menggunakan serveo maka buka tab terminal baru dan masukan command berikut
$ ssh -R 80:host lokal:3000 serveo.net
